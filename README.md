# Photography Agent

This is the frontend for the photography agent application. This is the code for the user interface that allows the user to chat, load conversations, and upload images. It contacts the backend via API.


## Workload split:
### Douglas Glover
Everything in the frontend was developed by Douglas Glover. The rest of the team worked with the backend.

## Local setup:

Run client and open in browser tab

npm run dev -- --open


Create a production version of the app (Later stuff)

npm run build
